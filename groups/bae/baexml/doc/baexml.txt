 baexml.txt

@PURPOSE: Provide utility components for using XML and XSD.

@MNEMONIC: Basic Application Environment XML (baexml)

@AUTHOR: Rohan Bhindwale (rbhindwa)

@SEE ALSO: a_xercesc, a_xml2

@DESCRIPTION: The 'baexml' package provides utilities for working with XML and
 XSD.  Clients can use the components provided by 'baexml' for parsing and
 printing fundamental types, XML schema, and XML data documents.

/Hierarchical Synopsis
/---------------------
 The 'baexml' package currently has 24 components having 8 levels of physical
 dependency.  The list below shows the hierarchical ordering of the components.
 The order of components within each level is not architecturally significant,

/Component Synopsis
/------------------
: 'baexml_base64parser':
:      Provide push parser for Base64 types.
:
: 'baexml_configschema':
:      Provide the 'baexml' schema
:
: 'baexml_datautil':
:      Utilities that produce bcem_Aggregate from XML data and XML schema
:
: 'baexml_decoder':
:      Provide a generic translation from XML into C++ objects.
:
: 'baexml_decoderoptions':
:      Provide value-semantic attribute classes
:
: 'baexml_element':
:      Provide a 'bcema_Blob'-based XML element container.
:
: 'baexml_elementattribute':
:      Provide the properties of an attribute in an XML element tag
:
: 'baexml_encoder':
:      Provide an XML encoder utility.
:
: 'baexml_encoderoptions':
:      Provide value-semantic attribute classes
:
: 'baexml_encodingstyle':
:      Provide value-semantic attribute classes
:
: 'baexml_errorinfo':
:      Provide common error information for XML components
:
: 'baexml_formatter':
:      Provide a simple interface for writing formatted XML.
:
: 'baexml_hexparser':
:      Provide push parser for hex types.
:
: 'baexml_listparser':
:      Provide push parser for lists.
:
: 'baexml_minireader':
:      Provide light-weight implementation of 'baexml_Reader' protocol
:
: 'baexml_namespaceregistry':
:      Provide a unique integer ID for each XML namespace.

:      Provide a unique integer ID for each XML namespace.

:

:      Provide a suite of common XML utilities.
