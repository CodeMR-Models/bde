 bdetu.txt

@PURPOSE: Provide utilities for working with 'bdet' types.

@MNEMONIC: Basic Development Environment 'bdeT' Utilities (bdetu)

@AUTHOR: Lea Fester (lfester)

@SEE_ALSO: bdet

@DESCRIPTION: The 'bdetu' package provides utility functions for use with
 'bdet' types.  In this release, these are predominantly time-related
 functions.

/Hierarchical Synopsis
/---------------------
 The 'bdetu' package currently has 12 components having 5 levels of physical
 dependency.  The list below shows the hierarchical ordering of the components.
 The order of components within each level is not architecturally significant,
 just alphabetical.
..
  5. bdetu_time

  4. bdetu_systemtime

  3. bdetu_datetime

  2. bdetu_epoch

  1. bdetu_date
     bdetu_datetimeinterval
     bdetu_dayofmonth
     bdetu_dayofweek
     bdetu_informixdatetime
     bdetu_month
     bdetu_timeinterval
     bdetu_unset
..

/Component Synopsis
/------------------
: 'bdetu_date':
:      Provide common non-primitive operations on 'bdet_Date'.
:
: 'bdetu_datetime':
:      Provide common non-primitive operations on 'bdet_Datetime'.
:
: 'bdetu_datetimeinterval':
:      Provide non-primitive procedures on 'bdet_DatetimeInterval'.
:
: 'bdetu_dayofmonth':
:      Provide non-primitive functionality on days of the month.
:
: 'bdetu_dayofweek':
:      Provide non-primitive functionality on days of the week.
:
: 'bdetu_epoch':
:      Conversion between absolute/relative time with respect to epoch.
:
: 'bdetu_informixdatetime':
:      Supply informix datetime operations on 'bdet_Datetime'.
:
: 'bdetu_month':
:      Provide non-primitive functionality on months of the year.
:
: 'bdetu_systemtime':
:      Provide utilities to retrieve the system time.
:
: 'bdetu_time':
:      Provide common non-primitive operations on 'bdet_Time'.
:
: 'bdetu_timeinterval':
:      Provide non-primitive procedures on 'bdet_TimeInterval'.
:
: 'bdetu_unset':
:      Define arbitrary "unset" values for fundamental and 'bdet' types.
