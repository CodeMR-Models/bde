@PURPOSE: Provide a mechanism to process Perl-compatible regular expressions.

@MNEMONIC: Basic Development Environment PCRE (bdepcre)

@AUTHOR: Arthur Chiu (achiu21)

@DESCRIPTION: This package provides a BDE-style mechanism that wraps the
 open-source PCRE package ('bde+pcre').  Users can create regular expressions
 compatible with Perl 5.8, and then match the regular expression to strings.

/Hierarchical Synopsis
/---------------------
 The 'bdepcre' package currently has 1 component.
..
  1. bdepcre_regex
..

/Component Synopsis
/------------------
: 'bdepcre_regex':
:      Provide a mechanism for regular expression pattern matching.
