 bdlblob.h

@PURPOSE: Provide flexible, efficient management of message buffers.

@MNEMONIC: Basic Development Library Blobs (bdlblob)

@AUTHOR: Steven Breitstein (sbreitstein)

@SEE_ALSO: bces, bcemt, bdema, bslma

@DESCRIPTION: This package defines, 'bdlblob::Blob', a vocabulary type for
 other 'bdl' packages.  A 'bdlblob::Blob' object consists of a sequence of
 'bdlblob::BlobBuffer' objects, each of which have a shared pointer to
 underlying memory.  A protocol is defined for factories to manage
 'bdlblob::BlobBuffer' objects, and a concrete implementation is provided.
 Other components define non-primitive operations on 'bdlblob::Blob's.

/Hierarchical Synopsis
/---------------------
 The 'bdlblob' package currently has 4 components having 2 levels of physical
 dependency.  The list below shows the hierarchical ordering of the components.
 The order of components within each level is not architecturally significant,
 just alphabetical.
..
  2. bdlblob_blobstreambuf
     bdlblob_blobutil
     bdlblob_pooledblobbufferfactory

  1. bdlblob_blob
..

/Component Synopsis
/------------------
: 'bdlblob_blob':
:      Provide an indexed set of buffers from multiple sources.
:
: 'bdlblob_blobstreambuf':
:      Provide blob implementing the 'streambuf' interface.
:
: 'bdlblob_blobutil':
:      Provide a suite of utilities for I/O operations on 'bdlblob::Blob'.
:
: 'bdlblob_pooledblobbufferfactory':
:      Provide a concrete implementation of 'bdlblob::BlobBufferFactory'.
