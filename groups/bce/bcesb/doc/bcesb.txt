@PURPOSE: Provide thread-enabled concrete 'streambuf' implementations.

@MNEMONIC: Basic Concurrency Environment Stream Buffers (bcesb)

@AUTHOR Tom Marshall (tmarshal)

@SEE_ALSO: bdesb

@DESCRIPTION: The 'bcesb' package provides concrete implementations of the
 'std::streambuf' interface.  In particular, these implementations are
 thread-enabled.

/Hierarchical Synopsis
/---------------------
 The 'bcesb' package currently has 2 components having 1 level of physical
 dependency.  The table below shows the hierarchical ordering of the
 components.  The order of components within each level is not architecturally
 significant, just alphabetical.
..
  1. bcesb_blobstreambuf
     bcesb_pooledbufferchainstreambuf
..

/Component Synopsis
/------------------
: 'bcesb_blobstreambuf':
:      Provide blob implementing the 'streambuf' interface.
:
: 'bcesb_pooledbufferchainstreambuf':
:      Provide pooled buffer chain implementing the 'streambuf' interface.
