BDE 3.33.1: Production Release¶

Schedule¶

  • The BDE team is pleased to announce that the BDE 3.33.1 production release was completed on Monday, July 15th, 2019

BDE 3.33.1 Highlights¶

New Component: bdlmt_signaler for connecting signals and slots¶

bdlmt::Signaler<PROT> has been contributed to BDE, a utility that provides an implementation of a signals and slots framework. This component provides a way to generate a function object (a signal) that can, in a thread safe manner, be attached to any number of compatible callable slots. Slots can be connected and disconnected from a signal asynchronously, and signals can be emitted concurrently.
